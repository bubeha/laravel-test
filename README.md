## Requirements

* PHP >= 8.2.0
* Laravel >= 10.0
* Docker >= 24
* Docker Compose >= 3

## Installation

```bash
// Clone source code.
$ git clone https://github.com/bubeha/laravel-test.git

// Copy .env.example to .env, and fill in the relevant configuration values.
$ cd laravel-test && cp .env.example .env

// Init application.
$ make init
```
Open http://localhost:8080/api/items
